# Tab-Time-Tracker
Tab Time Tracker
A Chrome extension that tracks how long you spend on each website throughout the day. It displays a live dashboard of your screen time per site, lets you set custom daily time limits, and sends a notification when you exceed your limit. All data is stored locally with no sign-up or API key needed.
How to install:

Download or clone this repository
Open Chrome and go to chrome://extensions
Enable Developer Mode (top right toggle)
Click "Load unpacked" and select the project folder

Built with: Vanilla JavaScript, Chrome Manifest V3
